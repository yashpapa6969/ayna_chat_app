# Flutter Chat App

This Flutter Chat App enables real-time communication between users through messages. Built with Flutter, the app leverages several packages for handling HTTP requests, real-time socket connections, local data storage, and the user interface.

## Features

- Real-time messaging
- User authentication
- Persistent message storage
- Customizable UI for chat
- Toast notifications

## Technologies Used

- Flutter
- Dart
- Node.js for backend services

## Packages

- `http`: ^1.2.1
- `fluttertoast`: ^8.2.5
- `socket_io_client`: ^2.0.3+1
- `shared_preferences`: ^2.2.3
- `flutter_chat_ui`: ^1.6.12
- `uuid`: any version

## Getting Started

### Prerequisites

Ensure you have Flutter installed by running:

```sh
flutter doctor
```

### Installation

1. **Clone the repository**:

    ```sh
    git clone  https://github.com/yashpapa6969/ayna_chat_app.git 
    cd cha_app
    ```

2. **Install dependencies**:

    ```sh
    flutter pub get
    ```

3. **Run the app**:

    ```sh
    flutter run
    ```

### Running the Backend Server

Assuming you have a Node.js server set up for handling the chat functionality:

1. **Navigate to your server directory** and install dependencies:

    ```sh
    cd path/to/your/server
    npm install
    ```

2. **Start the server**:

    ```sh
    npm start
    ```

## Usage

Once the Flutter app and backend server are up and running, users can sign up, log in, and start chatting in real time.
